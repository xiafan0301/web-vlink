# vlink-web

## Project setup
```
npm install
```

### Compiles and hot-reloads for development
```
npm run serve
npm run serve:dev
npm run serve:test
npm run serve:prod
```

### Compiles and minifies for production
```
npm run build
npm run build:dev
npm run build:test
npm run build:prod
```

### Run your tests
```
npm run test
```
